# Project Harness

This directory is a portable comparison harness that can be copied into any repository.

## Layout

- `原文档/docx/` and `原文档/pptx/`: source Office files used as the comparison index
- `standard/docx/` and `standard/pptx/`: reference PDFs exported from Word, PowerPoint, or WPS
- `target/docx/` and `target/pptx/`: PDFs generated by the project under evaluation
- `report/`: generated comparison outputs

Matching is based on relative path plus stem:

- `原文档/pptx/demo/slide.pptx`
- `standard/pptx/demo/slide.pdf`
- `target/pptx/demo/slide.pdf`

## Run

```bash
python3 project/compare_pdfs.py
```

Optional flags:

```bash
python3 project/compare_pdfs.py --project-root project --dpi 200
```

## Output

The script writes:

- `report/report.json`
- `report/report.md`
- `report/artifacts/<format>/<relative-stem>/`

Each artifact directory contains rendered page images, diff images, and extracted text.
